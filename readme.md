# Idle Game & Pixel Art Website

This project features a **fun and interactive idle mining game** alongside a **pixel-art-themed homepage**. The goal of the mining game is to collect resources, upgrade mining abilities, and manage money, while the website provides an engaging and modern design with SEO optimization and a layered animated background.

## Overview

The website is built with a pixel-art aesthetic and includes three main sections:

1. **Homepage:** A landing page with pixel-art backgrounds and navigation.
2. **About Page:** A simple informational page about the game.
3. **Idle Mining Game:** A fun, idle-style resource collection game where users can mine ores, upgrade their mining speed, and manage collected resources for profit.

## Features

### Website:
- **Pixel Art Design:** The website uses animated pixel-art backgrounds with multiple layers.
- **Random Background Selection:** A random background image is selected on each page load, creating a fresh experience every time.
- **Responsive Design:** The website is optimized for various screen sizes and devices.

### Idle Mining Game:
- **Resource Collection:** Collect different ores, each with its own mining speed and value.
- **Upgrade System:** Level up your mining abilities to increase the rate of resource collection.
- **Sell Ores for Money:** Exchange collected ores for money to continue upgrading.
- **Unlock New Ores:** Start with basic ores and unlock more valuable ores as you progress.
- **Progress Bars:** Visual representation of mining progress for each ore.
